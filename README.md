#### Для работы сборки у вас должны быть установлены
* npm 5
* gulp-cli

#### Как установить проект
1. clone this repo
2. npm i
3. gulp

#### О сборке gulpfile.js
Есть два таска js
1. C помощью плагинов gulp
2. С помощью gulp-webpack (закоментирован) entry point './app/js/app.js'
